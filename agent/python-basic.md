## 学习计划概览
> Python 本质还是一门编程语言，而编程语言的核心是3个：
> 1. 存储数据 — 在内存中记住某个值
> 2. 操作数据 — 对存储的值做运算（数学、逻辑、文本处理）
> 3. 控制流程 — 决定下一步做什么（条件、重复、跳转）
1. 存储数据：值与类型的表示 → 数据存储的最小单元
2. 操作数据：操作符 → 对数据进行操作
3. 控制流程：控制流语句 → 决定程序的走向
4. 复合应用
   - 数据结构：组织多个数据
   - 函数：操作数据和控制流程的抽象与复用
   - 输入与输出：与外部产生交互
   - 模块与包：基于函数及输入输出的更复合的组合
